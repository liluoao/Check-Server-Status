# Check Server Status

判断服务器状态

### 配置

- 添加你的服务器配置到`servers.xml`文件中：

#### 示例 :

```xml
<servers>
	<server id="42">
		<name>Home Server</name>
		<host>myserver.com</host>
		<port>80</port>
	</server>
</servers>
```
